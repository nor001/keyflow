# macOS manual follow-up

Use this after `install.sh`. Keep private/company data out of the repo.

## Must stay manual or validated first

- Comet: https://www.perplexity.ai/comet
- SAP GUI / SAP Logon: install through company/SAP channel.
- VMware Fusion: use Broadcom/VMware official portal if needed.
- Logi Options+: https://www.logitech.com/software/options.html
- Advanced keyboard/window automation tools for the future macOS version of keyflow.

## Why advanced automation tools are not auto-installed yet

They can change keyboard/input behavior. For the Mac migration, install and validate them manually first, then version their configuration only after the keyflow macOS architecture is stable.

## First-run checklist

```text
1. Restart terminal so PATH refreshes.
2. Run: git --version
3. Run: node --version
4. Run: python3 --version
5. Run: code --version
6. Run: cursor --version
7. Run: claude --version
8. Configure VS Code Settings Sync.
9. Configure Cursor login/settings.
10. Configure Obsidian vault.
11. Configure KeePassXC.
12. Configure Logi Options+ for MX Ergo.
13. Decide whether advanced keyboard/window automation becomes a keyflow/mac runtime dependency.
```
